enum CMR_SEYMOUR_DECOMPOSE_FLAG

Flags that indicate how to decompose as a \( 3 \)-sum.

See also
The desired types can be set by modifying CMR_SEYMOUR_PARAMS::decomposeStrategy.
Enumerator
CMR_SEYMOUR_DECOMPOSE_FLAG_DISTRIBUTED_MASK 

Bitmask for distributed rank treatment.

CMR_SEYMOUR_DECOMPOSE_FLAG_DISTRIBUTED_PIVOT 

Indicate to pivot a distributed rank distribution to a concentrated one;

See also
Decomposition of regular matroids.
CMR_SEYMOUR_DECOMPOSE_FLAG_DISTRIBUTED_DELTASUM 

Indicate to carry out a \( \Delta \)-sum for distributed ranks;

See also
Decomposition of regular matroids.
CMR_SEYMOUR_DECOMPOSE_FLAG_DISTRIBUTED_YSUM 

Indicate to carry out a Y-sum for distributed ranks;

See also
Decomposition of regular matroids.
CMR_SEYMOUR_DECOMPOSE_FLAG_CONCENTRATED_MASK 

Bitmask for distributed rank treatment.

CMR_SEYMOUR_DECOMPOSE_FLAG_CONCENTRATED_PIVOT 

Indicate to pivot a concentrated rank distribution to a distributed one;

See also
Decomposition of regular matroids.
CMR_SEYMOUR_DECOMPOSE_FLAG_CONCENTRATED_THREESUM 

Indicate to carry out a 3-sum for concentrated ranks;

See also
Decomposition of regular matroids.
CMR_SEYMOUR_DECOMPOSE_FLAG_SEYMOUR 

This triggers only \( \Delta \)-sum decompositions as defined by Seymour;

See also

CMR_SEYMOUR_DECOMPOSE_FLAG_TRUEMPER 

This triggers only 3-sum decompositions as defined by Truemper;

See also
Decomposition of regular matroids.

CMR_SEYMOUR_NODE **  clonedSubtrees 
)

Clones the union of subtrees of a Seymour decomposition, returning the copies.

The set of Seymour decomposition nodes that are (grand-)children of any of the numSubtrees nodes subtreeRoots is cloned. The respective clones are returned in clonedSubtrees.

◆ CMRseymourRelease()

CMR_EXPORT CMR_ERROR CMRseymourRelease ( CMR *  cmr,
CMR_SEYMOUR_NODE **  pnode 
)

Releases a decomposition node, freeing it if this was the last reference.

Decreases the reference counter by 1. If it reaches zero then it is freed. In that case, it is also called recursively for the child nodes. Sets *pnode to NULL to prevent accidental usage.

Parameters
cmrCMR environment.
pnodePointer to Seymour decomposition node. *pnode is set to NULL.
