Change Log
  • Revised the Seymour decomposition structure.
  • Fixed a bug in the code for printing dot-files for transposed graphic/network matrices; reported by Christopher Hojny.
  • Added a timeLimit parameter to all potentially time intensive functions.
  • Bugfix for the timeLimit parameter.
  • Added generator for wheel matrices.
  • Added first enumerative code for recognition of balanced matrices.

Version 1.3

  • Redesigned as Combinatorial Matrix Recognition Library.
  • Added recognition of (co)graphic and (co)network matrices.
  • Added recognition of series-parallel matrices.
  • Moved all C++ code to internal implementation. The library interface is only in C.
  • Added generators for network, graphic and random matrices.

TUtest Library

This library is the successor of the TUtest library, originally developed by Matthias Walter and Klaus Truemper for the recognition of totally unimodular matrices. The new name and additional functionality are available since version 1.3.
